Corpus Merger

A project, written in Java, working on very large data sets in S3. It extracted metadata from different formats of scientific articles and put it into bundles of JSON objects. To do this I utilized xPath.

Affiliation Disambiguation

A program which took affiliation declaration from publishers, matched them with an organization and gave them a unique research organization registry ID. The program, which is written in java, uses algorithms to preprocess the input strings and create queries, which are sent to elasticsearch. On return it sorts the results, estimating the chances of the match being good or bad. If it is bad, the program attempts the preprocessing and querying again, in a different way.

The end result is a JSON object, with the ID and how certain the program is of it being correct.

It was great fun researching for it and playing around with solutions, and I gladly talk about it in further detail with anyone who cares to listen.
